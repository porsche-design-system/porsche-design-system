# Media Query

The Porsche Design System provides SCSS Mixins for the media queries and SCSS Variables for breakpoints.

- min-width: `@mixin media-query-min('xs'|'sm'|'md'|'lg'|'xl'|'2xl')`
- max-width: `@mixin media-query-max('xs'|'sm'|'md'|'lg'|'xl'|'2xl')`
- min-width and max-width: `@mixin media-query-min-max('xs'|'sm'|'md'|'lg'|'xl'|'2xl', 'xs'|'sm'|'md'|'lg'|'xl'|'2xl')`.

```scss
// Usage example:
@include media-query-min('xs') {
  // styles for min-width: xs
}
```

All breakpoints are also available as SCSS Variables:

- xs: `$breakpoint-xs`
- sm: `$breakpoint-sm`
- md: `$breakpoint-md`
- lg: `$breakpoint-lg`
- xl: `$breakpoint-xl`
- 2xl: `$breakpoint-2xl`

Styles import: `@use '@porsche-design-system/components-{js|angular|react|vue}/scss' as *;`
