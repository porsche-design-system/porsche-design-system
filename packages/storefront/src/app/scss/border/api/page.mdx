## SCSS Variables

### Radius

| SCSS Variable  | Description                                | Value              |
| -------------- | ------------------------------------------ | ------------------ |
| `$radius-xs`   | Holds a **x-small** `border-radius`.       | {borderRadiusXs}   |
| `$radius-sm`   | Holds a **small** `border-radius`.         | {borderRadiusSm}   |
| `$radius-md`   | Holds a **medium** `border-radius`.        | {borderRadiusMd}   |
| `$radius-lg`   | Holds a **large** `border-radius`.         | {borderRadiusLg}   |
| `$radius-xl`   | Holds a **x-large** `border-radius`.       | {borderRadiusXl}   |
| `$radius-2xl`  | Holds a **2x-large** `border-radius`.      | {borderRadius2Xl}  |
| `$radius-3xl`  | Holds a **3x-large** `border-radius`.      | {borderRadius3Xl}  |
| `$radius-4xl`  | Holds a **4x-large** `border-radius`.      | {borderRadius4Xl}  |
| `$radius-full` | Holds a **fully** rounded `border-radius`. | {borderRadiusFull} |
