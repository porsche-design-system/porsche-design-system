## SCSS Variables

### Radius

| SCSS Variable  | Description                                | Value        |
| -------------- | ------------------------------------------ | ------------ |
| `$radius-xs`   | Holds a **x-small** `border-radius`.       | {radiusXs}   |
| `$radius-sm`   | Holds a **small** `border-radius`.         | {radiusSm}   |
| `$radius-md`   | Holds a **medium** `border-radius`.        | {radiusMd}   |
| `$radius-lg`   | Holds a **large** `border-radius`.         | {radiusLg}   |
| `$radius-xl`   | Holds a **x-large** `border-radius`.       | {radiusXl}   |
| `$radius-2xl`  | Holds a **2x-large** `border-radius`.      | {radius2Xl}  |
| `$radius-3xl`  | Holds a **3x-large** `border-radius`.      | {radius3Xl}  |
| `$radius-4xl`  | Holds a **4x-large** `border-radius`.      | {radius4Xl}  |
| `$radius-full` | Holds a **fully** rounded `border-radius`. | {radiusFull} |
