## SCSS Variables

### Radius

| SCSS Variable               | Description                         | Value            |
| --------------------------- | ----------------------------------- | ---------------- |
| `$pds-border-radius-small`  | Holds a **small** `border-radius`.  | {borderRadiusSm} |
| `$pds-border-radius-medium` | Holds a **medium** `border-radius`. | {borderRadiusMd} |
| `$pds-border-radius-large`  | Holds a **large** `border-radius`.  | {borderRadiusLg} |

### Width

| SCSS Variable            | Description                         | Value                |
| ------------------------ | ----------------------------------- | -------------------- |
| `$pds-border-width-thin` | Holds a **thin** `border-width`.    | {borderWidthThin}    |
| `$pds-border-width-base` | Holds a **regular** `border-width`. | {borderWidthRegular} |
