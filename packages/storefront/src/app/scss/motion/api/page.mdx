## SCSS Variables

### Duration

| SCSS variable         | Description                                  | Value                    |
| --------------------- | -------------------------------------------- | ------------------------ |
| `$motion-duration-sm` | Holds a **short** `transition-duration`.     | {motionDurationShort}    |
| `$motion-duration-md` | Holds a **moderate** `transition-duration`.  | {motionDurationModerate} |
| `$motion-duration-lg` | Holds a **long** `transition-duration`.      | {motionDurationLong}     |
| `$motion-duration-xl` | Holds a **very long** `transition-duration`. | {motionDurationVeryLong} |

### Ease

| SCSS variable         | Description                                       | Value             |
| --------------------- | ------------------------------------------------- | ----------------- |
| `$motion-ease-in-out` | Holds an **in-out** `transition-timing-function`. | {motionEaseInOut} |
| `$motion-ease-in`     | Holds an **in** `transition-timing-function`.     | {motionEaseIn}    |
| `$motion-ease-out`    | Holds an **out** `transition-timing-function`.    | {motionEaseOut}   |
