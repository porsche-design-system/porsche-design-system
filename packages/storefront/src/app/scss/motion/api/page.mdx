## SCSS Variables

### Duration

| SCSS variable  | Description                                                         | Value        |
| -------------- | ------------------------------------------------------------------- | ------------ |
| `$duration-sm` | Holds a **short** `transition-duration` / `animation-duration`.     | {durationSm} |
| `$duration-md` | Holds a **moderate** `transition-duration` / `animation-duration`.  | {durationMd} |
| `$duration-lg` | Holds a **long** `transition-duration` / `animation-duration`.      | {durationLg} |
| `$duration-xl` | Holds a **very long** `transition-duration` / `animation-duration`. | {durationXl} |

### Ease

| SCSS variable  | Description                                       | Value       |
| -------------- | ------------------------------------------------- | ----------- |
| `$ease-in-out` | Holds an **in-out** `transition-timing-function`. | {easeInOut} |
| `$ease-in`     | Holds an **in** `transition-timing-function`.     | {easeIn}    |
| `$ease-out`    | Holds an **out** `transition-timing-function`.    | {easeOut}   |
