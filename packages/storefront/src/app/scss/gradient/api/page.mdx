## SCSS Mixins

| SCSS Mixin                        | Description                                                       |
| --------------------------------- | ----------------------------------------------------------------- |
| `@mixin pds-gradient-to-top()`    | Applies a gradient faded to the **top** as `background-image`.    |
| `@mixin pds-gradient-to-right()`  | Applies a gradient faded to the **right** as `background-image`.  |
| `@mixin pds-gradient-to-bottom()` | Applies a gradient faded to the **bottom** as `background-image`. |
| `@mixin pds-gradient-to-left()`   | Applies a gradient faded to the **left** as `background-image`.   |
