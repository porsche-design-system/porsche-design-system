## SCSS Variables

### Fluid

| SCSS Variable        | Description                             | Value             |
| -------------------- | --------------------------------------- | ----------------- |
| `$spacing-fluid-xs`  | Applies the **x-small fluid** spacing.  | {spacingFluidXs}  |
| `$spacing-fluid-sm`  | Applies the **small fluid** spacing.    | {spacingFluidSm}  |
| `$spacing-fluid-md`  | Applies the **medium fluid** spacing.   | {spacingFluidMd}  |
| `$spacing-fluid-lg`  | Applies the **large fluid** spacing.    | {spacingFluidLg}  |
| `$spacing-fluid-xl`  | Applies the **x-large fluid** spacing.  | {spacingFluidXl}  |
| `$spacing-fluid-2xl` | Applies the **2x-large fluid** spacing. | {spacingFluid2Xl} |

### Static

| SCSS Variable         | Description                              | Value              |
| --------------------- | ---------------------------------------- | ------------------ |
| `$spacing-static-xs`  | Applies the **x-small static** spacing.  | {spacingStaticXs}  |
| `$spacing-static-sm`  | Applies the **small static** spacing.    | {spacingStaticSm}  |
| `$spacing-static-md`  | Applies the **medium static** spacing.   | {spacingStaticMd}  |
| `$spacing-static-lg`  | Applies the **large static** spacing.    | {spacingStaticLg}  |
| `$spacing-static-xl`  | Applies the **x-large static** spacing.  | {spacingStaticXl}  |
| `$spacing-static-2xl` | Applies the **2x-large static** spacing. | {spacingStatic2Xl} |
