## SCSS Variables

### Fluid

| SCSS Variable                 | Description                             | Value             |
| ----------------------------- | --------------------------------------- | ----------------- |
| `$pds-spacing-fluid-x-small`  | Applies the **x-small fluid** spacing.  | {spacingFluidXs}  |
| `$pds-spacing-fluid-small`    | Applies the **small fluid** spacing.    | {spacingFluidSm}  |
| `$pds-spacing-fluid-medium`   | Applies the **medium fluid** spacing.   | {spacingFluidMd}  |
| `$pds-spacing-fluid-large`    | Applies the **large fluid** spacing.    | {spacingFluidLg}  |
| `$pds-spacing-fluid-x-large`  | Applies the **x-large fluid** spacing.  | {spacingFluidXl}  |
| `$pds-spacing-fluid-xx-large` | Applies the **2x-large fluid** spacing. | {spacingFluid2Xl} |

### Static

| SCSS Variable                  | Description                              | Value              |
| ------------------------------ | ---------------------------------------- | ------------------ |
| `$pds-spacing-static-x-small`  | Applies the **x-small static** spacing.  | {spacingStaticXs}  |
| `$pds-spacing-static-small`    | Applies the **small static** spacing.    | {spacingStaticSm}  |
| `$pds-spacing-static-medium`   | Applies the **medium static** spacing.   | {spacingStaticMd}  |
| `$pds-spacing-static-large`    | Applies the **large static** spacing.    | {spacingStaticLg}  |
| `$pds-spacing-static-x-large`  | Applies the **x-large static** spacing.  | {spacingStaticXl}  |
| `$pds-spacing-static-xx-large` | Applies the **2x-large static** spacing. | {spacingStatic2Xl} |
