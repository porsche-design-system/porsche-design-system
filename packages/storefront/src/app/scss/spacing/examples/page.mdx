# Spacing

<TableOfContents headings={['Fluid', 'Static']} />

The Porsche Design System provides multiple SCSS variables for spacing styles.

Styles import: `@use '@porsche-design-system/components-{js|angular|react|vue}/scss' as *;`

## Fluid

Spacing scales dynamically based on the viewport width, within defined minimum and maximum limits. Mainly intended for
**vertical** spacing between elements.

<ComponentStory story={spacingFluidScssStory} showCodeBlock={false} />

## Static

Spacing uses static, pixel-based values. Primarily used for **horizontal** spacing between elements.
