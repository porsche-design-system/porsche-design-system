## SCSS Mixins

| SCSS Mixin                        | Description                        |
| --------------------------------- | ---------------------------------- |
| `@mixin pds-drop-shadow-high()`   | Applies a **high** `box-shadow`.   |
| `@mixin pds-drop-shadow-medium()` | Applies a **medium** `box-shadow`. |
| `@mixin pds-drop-shadow-low()`    | Applies a **low** `box-shadow`.    |
