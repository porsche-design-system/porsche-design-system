## Setup

The Porsche Design System includes recommended **normalize styles**, which provide:

- CSS reset styles
- Base typography definitions

These styles are included in the main CSS file:

```css
@import '@porsche-design-system/components-{js|angular|react|vue}/index.css';
```

Alternatively, you can import only the normalize styles:

```css
@import '@porsche-design-system/components-{js|angular|react|vue}/normalize.css';
```
