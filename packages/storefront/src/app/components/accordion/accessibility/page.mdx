## Accessibility support

### Keyboard

| Key / state        | Function                                                                           |
| ------------------ | ---------------------------------------------------------------------------------- |
| `Tab`, `Shift-Tab` | Moves focus to the next (or previous) focusable element.                           |
| `Enter/Space`      | When focus is on the accordion trigger, the associated panel expands or collapses. |

## Development considerations

### Heading hierarchy

The accordion summary should be a heading element (e.g.,`p-heading` or `<h1>` to `<h6>`,) to ensure proper semantic
structure and correct heading hierarchy.
