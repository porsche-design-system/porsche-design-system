## Sticky headline

The headline can be made sticky by adding the property `sticky` to the `p-accordion` tag. The position `top` value can be overwritten by using the CSS Custom Property `--p-accordion-position-sticky-top`.

## Custom clickable/focusable area

Sometimes it might be useful to enlarge the clickable/focusable area to fulfill accessibility guidelines. Therefore, a
custom padding can be set on the slotted heading element.
