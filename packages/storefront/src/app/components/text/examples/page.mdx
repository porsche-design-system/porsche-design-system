## Hyphens

By default, the `p-text` component has hyphens set to `auto`, but this can be overridden using the `--p-hyphens` CSS
variable. When inserting manual hyphens, you can choose to use soft hyphens. For example, `extra&shy;ordinarily`, as
shown in the second example.

However, please note that hyphenation behavior can vary depending on the browser and the language of the text. In some
cases, it may be necessary to set the appropriate lang attribute on your HTML element to ensure that hyphenation works
correctly for the desired language.
