## Slotted button

The `p-popover` component can also be used with a **slotted button** via the `button` slot, allowing you to provide a
custom button element.

<ComponentStory story={popoverSlottedButtonStory} />

### The following development considerations must be taken into account

1. The `aria-expanded` attribute must be set on the custom button element and changed manually depending on the
   popover's open state to `true`/`false`.
