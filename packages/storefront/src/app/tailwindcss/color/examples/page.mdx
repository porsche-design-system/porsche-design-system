# Color

<TableOfContents headings={['Text Color', 'Background Color', 'Border Color']} />

The Porsche Design System Tailwind theme customizes Tailwind variables such as `--color-*`, which affect multiple
Tailwind classes including `bg-*`, `text-*`, `border-*` and others.

The colors use the
[light-dark()](https://developer.mozilla.org/en-US/docs/Web/CSS/Reference/Values/color_value/light-dark) CSS color
function. To change the theme, use the utility classes `.scheme-light`, `.scheme-dark` or `.scheme-light-dark`. Learn
more in the [Color Scheme](/tailwindcss/introduction#color-scheme) section.
