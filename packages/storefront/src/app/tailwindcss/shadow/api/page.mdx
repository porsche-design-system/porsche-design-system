## Variables

| Tailwind CSS variable | Tailwind CSS class | Description                        |
| --------------------- | ------------------ | ---------------------------------- |
| `--shadow-lg`         | `.shadow-lg`       | Applies a **large** `box-shadow`.  |
| `--shadow-md`         | `.shadow-md`       | Applies a **medium** `box-shadow`. |
| `--shadow-sm`         | `.shadow-sm`       | Applies a **small** `box-shadow`.  |
