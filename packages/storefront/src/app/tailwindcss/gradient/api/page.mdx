# Gradient

<TableOfContents headings={[]} />

## Classes

| Tailwind `@utility` CSS class | Description                                                       |
| ----------------------------- | ----------------------------------------------------------------- |
| `.bg-fade-to-t`               | Applies a gradient faded to the **top** as `background-image`.    |
| `.bg-fade-to-r`               | Applies a gradient faded to the **right** as `background-image`.  |
| `.bg-fade-to-b`               | Applies a gradient faded to the **bottom** as `background-image`. |
| `.bg-fade-to-l`               | Applies a gradient faded to the **left** as `background-image`.   |
