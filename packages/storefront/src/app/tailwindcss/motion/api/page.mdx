## Variables

| CSS variable                      | Tailwind CSS class    | Description                                         |
| --------------------------------- | --------------------- | --------------------------------------------------- |
| `--transition-duration-short`     | `.duration-short`     | Applies a **short** `transition-duration`.          |
| `--transition-duration-moderate`  | `.duration-moderate`  | Applies a **moderate** `transition-duration`.       |
| `--transition-duration-long`      | `.duration-long`      | Applies a **long** `transition-duration`.           |
| `--transition-duration-very-long` | `.duration-very-long` | Applies a **very long** `transition-duration`.      |
| `--ease-in-out`                   | `.ease-in-out`        | Applies an **in-out** `transition-timing-function`. |
| `--ease-in`                       | `.ease-in`            | Applies an **in** `transition-timing-function`.     |
| `--ease-out`                      | `.ease-out`           | Applies an **out** `transition-timing-function`.    |
