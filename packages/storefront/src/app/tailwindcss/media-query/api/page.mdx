## Variables

| Tailwind CSS variable | Tailwind CSS class                    | Description                                                   |
| --------------------- | ------------------------------------- | ------------------------------------------------------------- |
| `--breakpoint-xs`     | `.xs:*`, `.max-xs:*`                  | Applies a media query with the `xs` (**480px**) breakpoint.   |
| `--breakpoint-sm`     | `.sm:*`, `.max-sm:*`, `.xs:max-sm:*`  | Applies a media query with the `sm` (**760px**) breakpoint.   |
| `--breakpoint-md`     | `.md:*`, `.max-md:*`, `.md:max-lg:*`  | Applies a media query with the `md` (**1000px**) breakpoint.  |
| `--breakpoint-lg`     | `.lg:*`, `.max-lg:*`, `.lg:max-xl:*`  | Applies a media query with the `lg` (**1300px**) breakpoint.  |
| `--breakpoint-xl`     | `.xl:*`, `.max-xl:*`, `.xl:max-2xl:*` | Applies a media query with the `xl` (**1760px**) breakpoint.  |
| `--breakpoint-2xl`    | `.2xl:*`, `.max-2xl:*`                | Applies a media query with the `2xl` (**1920px**) breakpoint. |
