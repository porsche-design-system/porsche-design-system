## Vanilla Extract

These styles are built on top of the design tokens from the [Tokens](/tokens/introduction) package. If you use a
different styling solution, you can consume the tokens directly.

## ðŸ‘¹Breaking Changes

- Changed import path for Vanilla Extract styles.

```diff
- import { â€¦ } from '@porsche-design-system/components-{js|angular|react|vue}/styles/vanilla-extract';
+ import { â€¦ } from '@porsche-design-system/components-{js|angular|react|vue}/vanilla-extract';
```

- Removed `getHoverStyle` since it did not work reliable.

- Removed `opts` parameter in `getSkeletonStyle()` including `theme` (New skeleton style works with `light-dark()` CSS
  color function)

```diff
const skeletonAnimation = keyframes(skeletonKeyframes);
- const Skeleton = style(getSkeletonStyle(skeletonAnimation, { theme: 'light|dark' }));
+ const Skeleton = style(getSkeletonStyle(skeletonAnimation));
```

## Upcoming Changes

All following old styles are still available but marked as deprecated and will be removed with the next major release.
Please switch to the new styles as soon as possible.
