## Migrate from Porsche Design System v3 to v4

With the new **Porsche Design Language** comes a lot of changes regarding layout and design principles. To keep
refactoring efforts as low as possible when upgrading from `v3` to `v4`, **breaking changes** were avoided as far as
possible. Nevertheless, there are a few breaking changes and some more deprecations which should receive attention.

Please follow the migration guide on this page. In addition, we offer a [components overview](/components/introduction)
which shows all component information including deprecated components/props/values.

<Notification heading="Deprecation hint" headingTag="h3" state="error">
  Porsche Design System **v3** support will end on **TBD** (until EOL date only critical bug fixing takes place).

**Please make sure that upgrading to Porsche Design System v4 is feasible before the EOL date.**

</Notification>

## Notable New Features

- New design language for all components
- All components use CSS Custom Properties
- Theming now works by adding a `light`, `dark` or `auto` class to any HTML element

## üõ†Ô∏è Setup Changes

- Removed `getInitialStyles()`, `getFontFaceStyles()` and `getFontFaceStylesheet()` partial in favor of CSS imports from
  `@porsche-design-system/components-{js|angular|react|vue}`.

```diff
- import { getInitialStyles, getFontFaceStyles, getFontFaceStylesheet } from '@porsche-design-system/components-{js|angular|react|vue}/partials';
```

- Instead, a mandatory CSS global style, including CSS color variables, font-face definitions and normalize styles needs
  to be imported into your main css file. See [Global Styles](/global-styles/introduction/) for more information.

```diff
/* src/style.css */
+ @import '@porsche-design-system/components-{js|angular|react|vue}/index.css';
/* Alternative with fonts from China CDN */
+ @import '@porsche-design-system/components-{js|angular|react|vue}/cn/index.css';
```

- Automatic FOUC handling, previously defined in `getInitialStyles()`, must be implemented manually now. See Getting
  Started in the [Development](/developing/introduction/) section for more information.

```diff
+ :not(:defined) {
+   visibility: hidden;
+ }
```

- Removed `getBrowserSupportFallbackScript()` and `getCookiesFallbackScript()` which are now in separate repo
  `@porsche-design-system/fallbacks` coming soon.

```diff
- import { getBrowserSupportFallbackScript, getCookiesFallbackScript } from '@porsche-design-system/components-{js|angular|react|vue}/partials';
```

- Removed `getDSRPonyfill()` partial as it is no longer needed.

```diff
- import { getDSRPonyfill } from '@porsche-design-system/components-{js|angular|react|vue}/partials';
```

