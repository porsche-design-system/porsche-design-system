# Header

<TableOfContents headings={['Variant 1', 'Variant 2']} />

The header serves as the primary navigation and identification element at the top of each page. It provides users with
consistent access to key functionalities, site-wide navigation, and brand recognition. This section outlines the
structure, variations, and usage guidelines for the header component, ensuring a unified and intuitive experience across
all digital touchpoints. Understanding and implementing the header correctly is crucial for establishing a clear and
efficient user journey.
