# Typography

<TableOfContents headings={['Classes', 'Variables']} />

## JS

### Display

| JS                   | Description                                                                        |
| -------------------- | ---------------------------------------------------------------------------------- |
| `displayLargeStyle`  | Applies the **large** variant of extra-large typography primarily to `<h1>` tags.  |
| `displayMediumStyle` | Applies the **medium** variant of extra-large typography primarily to `<h1>` tags. |
| `displaySmallStyle`  | Applies the **small** variant of extra-large typography primarily to `<h1>` tags.  |

### Heading

| JS                    | Description                                                                                                           |
| --------------------- | --------------------------------------------------------------------------------------------------------------------- |
| `headingXXLargeStyle` | Applies the **2x-large** heading typography variant primarily to `<h1>`, `<h2>`, `<h3>`, `<h4>`, `<h5>`, `<h6>` tags. |
| `headingXLargeStyle`  | Applies the **x-large** heading typography variant primarily to `<h1>`, `<h2>`, `<h3>`, `<h4>`, `<h5>`, `<h6>` tags.  |
| `headingLargeStyle`   | Applies the **large** heading typography variant primarily to `<h1>`, `<h2>`, `<h3>`, `<h4>`, `<h5>`, `<h6>` tags.    |
| `headingMediumStyle`  | Applies the **medium** heading typography variant primarily to `<h1>`, `<h2>`, `<h3>`, `<h4>`, `<h5>`, `<h6>` tags.   |
| `headingSmallStyle`   | Applies the **small** heading typography variant primarily to `<h1>`, `<h2>`, `<h3>`, `<h4>`, `<h5>`, `<h6>` tags.    |

### Text

| JS                 | Description                                                                                               |
| ------------------ | --------------------------------------------------------------------------------------------------------- |
| `textXLargeStyle`  | Applies the **x-large** text typography variant primarily to `<p>`, `<ul>`, `<ol>`, `<blockquote>` tags.  |
| `textLargeStyle`   | Applies the **large** text typography variant primarily to `<p>`, `<ul>`, `<ol>`, `<blockquote>` tags.    |
| `textMediumStyle`  | Applies the **medium** text typography variant primarily to `<p>`, `<ul>`, `<ol>`, `<blockquote>` tags.   |
| `textSmallStyle`   | Applies the **small** text typography variant primarily to `<p>`, `<ul>`, `<ol>`, `<blockquote>` tags.    |
| `textXSmallStyle`  | Applies the **x-small** text typography variant primarily to `<p>`, `<ul>`, `<ol>`, `<blockquote>` tags.  |
| `textXXSmallStyle` | Applies the **2x-small** text typography variant primarily to `<p>`, `<ul>`, `<ol>`, `<blockquote>` tags. |
