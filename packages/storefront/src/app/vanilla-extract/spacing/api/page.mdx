## JS

### Spacing

| JS        | Description                                              | Value                     |
| --------- | -------------------------------------------------------- | ------------------------- |
| `spacing` | Object containing all **fluid** and **static** spacings. | {JSON.stringify(spacing)} |

### Fluid

| JS                    | Description                               | Value                          |
| --------------------- | ----------------------------------------- | ------------------------------ |
| `spacingFluid`        | Object containing all **fluid** spacings. | {JSON.stringify(spacingFluid)} |
| `spacingFluidXSmall`  | Applies the **x-small fluid** spacing.    | {spacingFluidXSmall}           |
| `spacingFluidSmall`   | Applies the **small fluid** spacing.      | {spacingFluidSmall}            |
| `spacingFluidMedium`  | Applies the **medium fluid** spacing.     | {spacingFluidMedium}           |
| `spacingFluidLarge`   | Applies the **large fluid** spacing.      | {spacingFluidLarge}            |
| `spacingFluidXLarge`  | Applies the **x-large fluid** spacing.    | {spacingFluidXLarge}           |
| `spacingFluidXXLarge` | Applies the **2x-large fluid** spacing.   | {spacingFluidXXLarge}          |
