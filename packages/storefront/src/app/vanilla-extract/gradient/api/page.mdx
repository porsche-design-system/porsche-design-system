# Gradient

<TableOfContents headings={[]} />

## JS

| JS                      | Description                                                       |
| ----------------------- | ----------------------------------------------------------------- |
| `gradientToTopStyle`    | Applies a gradient faded to the **top** as `background-image`.    |
| `gradientToRightStyle`  | Applies a gradient faded to the **right** as `background-image`.  |
| `gradientToBottomStyle` | Applies a gradient faded to the **bottom** as `background-image`. |
| `gradientToLeftStyle`   | Applies a gradient faded to the **left** as `background-image`.   |
