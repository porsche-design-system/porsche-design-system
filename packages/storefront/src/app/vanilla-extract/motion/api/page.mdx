## JS

### Duration

| JS           | Description                                                         | Value        |
| ------------ | ------------------------------------------------------------------- | ------------ |
| `durationSm` | Holds a **short** `transition-duration` / `animation-duration`.     | {durationSm} |
| `durationMd` | Holds a **moderate** `transition-duration` / `animation-duration`.  | {durationMd} |
| `durationLg` | Holds a **long** `transition-duration` / `animation-duration`.      | {durationLg} |
| `durationXl` | Holds a **very long** `transition-duration` / `animation-duration`. | {durationXl} |

### Ease

| JS          | Description                                       | Value       |
| ----------- | ------------------------------------------------- | ----------- |
| `easeInOut` | Holds an **in-out** `transition-timing-function`. | {easeInOut} |
| `easeIn`    | Holds an **in** `transition-timing-function`.     | {easeIn}    |
| `easeOut`   | Holds an **out** `transition-timing-function`.    | {easeOut}   |
