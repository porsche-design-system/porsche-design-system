# Media Query

The Porsche Design System provides JS Functions for the media queries and JS Variables for breakpoints to be used in
Vanilla Extract.

- min-width: `getMediaQueryMin('xs'|'sm'|'md'|'lg'|'xl'|'2xl')`
- max-width: `getMediaQueryMax('xs'|'sm'|'md'|'lg'|'xl'|'2xl')`
- min-width and max-width: `getMediaQueryMinMax('xs'|'sm'|'md'|'lg'|'xl'|'2xl', 'xs'|'sm'|'md'|'lg'|'xl'|'2xl')`.

```js
// Usage example:
const MediaQueryMin = style({
  '@media': {
    [getMediaQueryMin('xs')]: {
      // styles for min-width: xs
    },
  },
});
```

All breakpoints are also available as JS Variables:

- xs: `breakpointXs`
- sm: `breakpointSm`
- md: `breakpointMd`
- lg: `breakpointLg`
- xl: `breakpointXl`
- 2xl: `breakpoint2XL`
