# Media Query

<TableOfContents headings={['JS Functions', 'JS']} />

## JS Functions

| JS Function                                                                                     | Description                                                       |
| ----------------------------------------------------------------------------------------------- | ----------------------------------------------------------------- |
| `getMediaQueryMin('xs'\|'sm'\|'md'\|'lg'\|'xl'\|'2xl')`                                         | Applies a **min** media query with the specified breakpoint.      |
| `getMediaQueryMax('xs'\|'sm'\|'md'\|'lg'\|'xl'\|'2xl')`                                         | Applies a **max** media query with the specified breakpoint.      |
| `getMediaQueryMinMax('xs'\|'sm'\|'md'\|'lg'\|'xl'\|'2xl', 'xs'\|'sm'\|'md'\|'lg'\|'xl'\|'2xl')` | Applies a **min-max** media query with the specified breakpoints. |

## JS

| JS              | Description                       | Value           |

| `breakpointLg`  | Holds the `lg` breakpoint value.  | {breakpointLg}  |
| `breakpointXl`  | Holds the `xl` breakpoint value.  | {breakpointXl}  |
