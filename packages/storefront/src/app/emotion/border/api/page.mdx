## JS

### Radius

| JS           | Description                                | Value        |
| ------------ | ------------------------------------------ | ------------ |
| `radiusXs`   | Holds a **x-small** `border-radius`.       | {radiusXs}   |
| `radiusSm`   | Holds a **small** `border-radius`.         | {radiusSm}   |
| `radiusMd`   | Holds a **medium** `border-radius`.        | {radiusMd}   |
| `radiusLg`   | Holds a **large** `border-radius`.         | {radiusLg}   |
| `radiusXl`   | Holds a **x-large** `border-radius`.       | {radiusXl}   |
| `radius2Xl`  | Holds a **2x-large** `border-radius`.      | {radius2Xl}  |
| `radius3Xl`  | Holds a **3x-large** `border-radius`.      | {radius3Xl}  |
| `radius4Xl`  | Holds a **4x-large** `border-radius`.      | {radius4Xl}  |
| `radiusFull` | Holds a **fully** rounded `border-radius`. | {radiusFull} |
