## JS

### Duration

| JS                       | Description                                  | Value                    |
| ------------------------ | -------------------------------------------- | ------------------------ |
| `motionDurationShort`    | Holds a **short** `transition-duration`.     | {motionDurationShort}    |
| `motionDurationModerate` | Holds a **moderate** `transition-duration`.  | {motionDurationModerate} |
| `motionDurationLong`     | Holds a **long** `transition-duration`.      | {motionDurationLong}     |
| `motionDurationVeryLong` | Holds a **very long** `transition-duration`. | {motionDurationVeryLong} |

### Ease

| JS                 | Description                                       | Value              |
| ------------------ | ------------------------------------------------- | ------------------ |
| `motionEasingBase` | Holds an **in-out** `transition-timing-function`. | {motionEasingBase} |
| `motionEasingIn`   | Holds an **in** `transition-timing-function`.     | {motionEasingIn}   |
| `motionEasingOut`  | Holds an **out** `transition-timing-function`.    | {motionEasingOut}  |
