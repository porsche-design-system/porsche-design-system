## JS

### Duration

| JS           | Description                                  | Value        |
| ------------ | -------------------------------------------- | ------------ |
| `durationSm` | Holds a **short** `transition-duration`.     | {durationSm} |
| `durationMd` | Holds a **moderate** `transition-duration`.  | {durationMd} |
| `durationLg` | Holds a **long** `transition-duration`.      | {durationLg} |
| `durationXl` | Holds a **very long** `transition-duration`. | {durationXl} |

### Ease

| JS                 | Description                                       | Value              |
| ------------------ | ------------------------------------------------- | ------------------ |
| `motionEasingBase` | Holds an **in-out** `transition-timing-function`. | {motionEasingBase} |
| `motionEasingIn`   | Holds an **in** `transition-timing-function`.     | {motionEasingIn}   |
| `motionEasingOut`  | Holds an **out** `transition-timing-function`.    | {motionEasingOut}  |
