# Media Query

<TableOfContents headings={[]} />

| Tailwind CSS variable | Tailwind CSS class                 | Description                                                                 |
| --------------------- | ---------------------------------- | --------------------------------------------------------------------------- |
| `--breakpoint-xs`     | `.xs:*`, `.max-xs:*`               | Applies the style at or below the `xs` breakpoint.                          |
| `--breakpoint-sm`     | `.sm:*`, `.max-sm:*`, `xs:max-sm`  | Applies the style at or above the `sm` breakpoint, or within the `sm` range |
| `--breakpoint-md`     | `.md:*`, `.max-md:*`, `md:max-lg`  | Applies the style at or above the `md` breakpoint, or within the `md` range |
| `--breakpoint-lg`     | `.lg:*`, `.max-lg:*`, `lg:max-xl`  | Applies the style at or above the `lg` breakpoint, or within the `lg` range |
| `--breakpoint-xl`     | `.xl:*`, `.max-xl:*`, `xl:max-2xl` | Applies the style at or above the `xl` breakpoint, or within the `xl` range |
| `--breakpoint-2xl`    | `.2xl:*`, `.max-2xl:*`             | Applies the style at or above the `2xl` breakpoint                          |
