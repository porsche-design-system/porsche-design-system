# Border

<TableOfContents headings={['Border Radius', 'Border Width']} />

## Border Radius

| Tailwind CSS variable | Tailwind CSS class | Description                      |
| --------------------- | ------------------ | -------------------------------- |
| `--radius-sm`         | `.rounded-sm`      | Applied a small `border-radius`  |
| `--radius-md`         | `.rounded-md`      | Applied a medium `border-radius` |
| `--radius-lg`         | `.rounded-lg`      | Applied a large `border-radius`  |

## Border Width

| Tailwind CSS variable    | Tailwind CSS class | Description                        |
| ------------------------ | ------------------ | ---------------------------------- |
| `--default-border-width` | `.border`          | Applies the default `border-width` |
| `--border-width-thin`    | `.border-thin`     | Applies a thinner `border-width`   |
