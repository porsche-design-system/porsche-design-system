# Border

<TableOfContents headings={["Border Radius","Border Width"]} />

## Border Radius

| Tailwind CSS variable | Tailwind CSS class | Description                           |
| --------------------- | ------------------ | ------------------------------------- |
| `--radius-sm`         | `.rounded-sm`      | Applies a **small** `border-radius`.  |
| `--radius-md`         | `.rounded-md`      | Applies a **medium** `border-radius`. |
| `--radius-lg`         | `.rounded-lg`      | Applies a **large** `border-radius`.  |

## Border Width

| Tailwind CSS variable    | Tailwind CSS class | Description                             |
| ------------------------ | ------------------ | --------------------------------------- |
| `--default-border-width` | `.border`          | Applies the **default** `border-width`. |
| `--border-width-thin`    | `.border-thin`     | Applies a **thin** `border-width`.      |
