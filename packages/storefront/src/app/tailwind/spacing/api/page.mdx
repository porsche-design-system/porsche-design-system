## Fluid

| Tailwind CSS variable | Tailwind CSS class              | Description                                 |
| --------------------- | ------------------------------- | ------------------------------------------- |
| `--spacing-fluid-xs`  | `p-fluid-xs`, `m-fluid-xs`...   | Applies the extra small fluid spacing       |
| `--spacing-fluid-sm`  | `p-fluid-sm`, `m-fluid-sm`...   | Applies the small fluid spacing             |
| `--spacing-fluid-md`  | `p-fluid-md`, `m-fluid-md`...   | Applies the medium fluid spacing            |
| `--spacing-fluid-lg`  | `p-fluid-lg`, `m-fluid-lg`...   | Applies the large fluid spacing             |
| `--spacing-fluid-xl`  | `p-fluid-xl`, `m-fluid-xl`...   | Applies the extra large fluid spacing       |
| `--spacing-fluid-2xl` | `p-fluid-2xl`, `m-fluid-2xl`... | Applies the extra extra large fluid spacing |
