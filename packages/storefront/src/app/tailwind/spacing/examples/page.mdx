# Spacing

<TableOfContents headings={["Fluid","Static"]} />

The Porsche Design System Tailwind theme customizes the Tailwind `--spacing-*` CSS variables. These variables are used
by various Tailwind utility classes, such as `.p-fluid-*`, `.m-static-*`, and others.

Standard Tailwind spacing utilities like `.p-2` will continue to work as expected. Additionally, we also provide custom
spacing values available in fluid or static and various sizes.
