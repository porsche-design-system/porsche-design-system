## Variables

| Tailwind CSS variable | Tailwind CSS class | Description                        |
| --------------------- | ------------------ | ---------------------------------- |
| `--shadow-high`       | `.shadow-high`     | Applies a **high** `box-shadow`.   |
| `--shadow-medium`     | `.shadow-medium`   | Applies a **medium** `box-shadow`. |
| `--shadow-low`        | `.shadow-low`      | Applies a **low** `box-shadow`.    |
