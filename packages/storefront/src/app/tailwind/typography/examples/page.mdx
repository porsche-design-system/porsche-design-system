# Typography

<TableOfContents headings={['Display', 'Heading', 'Text']} />

The Porsche Design System Tailwind theme includes the custom `@utility` CSS classes `.pds-display-*`, `.pds-heading-*`
and `.pds-text-*` to style headings and paragraphs consistently.

Additionally, it customizes Tailwind variables such as `--font-*`, `--leading-*` and `--text-*`, which affect multiple
Tailwind classes including `.font-*`, `.leading-*`, `.text-*` and others.
