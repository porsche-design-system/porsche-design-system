# Typography

<TableOfContents headings={["Display","Heading","Text"]} />

The Porsche Design System Tailwind theme exposes custom Tailwind `@utility` CSS classes, `.pds-display-*`,
`.pds-heading-*` and `.pds-text-*` to properly style headings and paragraphs.

In addition, Tailwind theme variables `--font-*`, `--leading-*` and `--text-*` are customized. This will affect multiple
utility classes like `.font-*`, `.leading-*`, `.text-*`, etc.
