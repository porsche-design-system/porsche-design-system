# Gradient

<TableOfContents headings={[]} />

| Tailwind `@utility` CSS class | Description                                              |
| ----------------------------- | -------------------------------------------------------- |
| `.bg-fade-to-b`               | Applies a gradient for `background-image` to the bottom. |
| `.bg-fade-to-l`               | Applies a gradient for `background-image` to the left.   |
| `.bg-fade-to-r`               | Applies a gradient for `background-image` to the right.  |
| `.bg-fade-to-t`               | Applies a gradient for `background-image` to the top.    |
