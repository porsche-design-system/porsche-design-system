# Gradient

<TableOfContents headings={[]} />

| Tailwind CSS variable  | Tailwind CSS class                                                          | Description                                              |
| ---------------------- | --------------------------------------------------------------------------- | -------------------------------------------------------- |
| `--gradient-to-bottom` | `.bg-linear-(--gradient-to-bottom)`, `.bg-radial-(--gradient-to-bottom)`... | Applies a gradient for `background-image` to the bottom. |
| `--gradient-to-left`   | `.bg-linear-(--gradient-to-left)`, `.bg-radial-(--gradient-to-left)`...     | Applies a gradient for `background-image` to the left.   |
| `--gradient-to-right`  | `.bg-linear-(--gradient-to-right)`, `.bg-radial-(--gradient-to-right)`...   | Applies a gradient for `background-image` to the right.  |
| `--gradient-to-top`    | `.bg-linear-(--gradient-to-top)`, `.bg-radial-(--gradient-to-top)`...       | Applies a gradient for `background-image` to the top.    |
