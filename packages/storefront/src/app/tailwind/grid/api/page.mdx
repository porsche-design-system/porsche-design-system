# Grid

<TableOfContents headings={["Classes","Variables"]} />

## Classes

| Tailwind `@utility` CSS class | Description                                                                                                                         |
| ----------------------------- | ----------------------------------------------------------------------------------------------------------------------------------- |
| `.pds-grid`                   | Applies the Porsche Grid layout system (must be applied once at the top level, span the full viewport width, and cannot be nested). |

## Variables

| Tailwind CSS variable               | Tailwind CSS class                              | Description                                                              |
| ----------------------------------- | ----------------------------------------------- | ------------------------------------------------------------------------ |
| `--pds-grid-full-column-start`      | `.col-start-(--pds-grid-full-column-start)`     | Applies the start position of the `full` column in the Porsche Grid.     |
| `--pds-grid-full-column-end`        | `.col-end-(--pds-grid-full-column-end)`         | Applies the end position of the `full` column in the Porsche Grid.       |
| `--pds-grid-wide-column-start`      | `.col-start-(--pds-grid-wide-column-start)`     | Applies the start position of the `wide` column in the Porsche Grid.     |
| `--pds-grid-wide-column-end`        | `.col-end-(--pds-grid-wide-column-end)`         | Applies the end position of the `wide` column in the Porsche Grid.       |
| `--pds-grid-extended-column-start`  | `.col-start-(--pds-grid-extended-column-start)` | Applies the start position of the `extended` column in the Porsche Grid. |
| `--pds-grid-extended-column-end`    | `.col-end-(--pds-grid-extended-column-end)`     | Applies the end position of the `extended` column in the Porsche Grid.   |
| `--pds-grid-extended-span-one-half` | `.col-(pds-grid-extended-span-one-half)`        | Applies a half span within the `extended` area of the Porsche Grid.      |
| `--pds-grid-basic-column-start`     | `.col-start-(--pds-grid-basic-column-start)`    | Applies the start position of the `basic` column in the Porsche Grid.    |

| `--pds-grid-basic-span-one-half`    | `.col-(pds-grid-basic-span-one-half)`           | Applies a half span within the `basic` area of the Porsche Grid.         |
| `--pds-grid-basic-span-one-third`   | `.col-(pds-grid-basic-span-one-third)`          | Applies a one third span within the `basic` area of the Porsche Grid.    |
| `--pds-grid-basic-span-two-thirds`  | `.col-(pds-grid-basic-span-two-thirds)`         | Applies a two thirds span within the `basic` area of the Porsche Grid.   |
| `--pds-grid-narrow-column-start`    | `.col-start-(--pds-grid-narrow-column-start)`   | Applies the start position of the `narrow` column in the Porsche Grid.   |
| `--pds-grid-narrow-column-end`      | `.col-end-(--pds-grid-narrow-column-end)`       | Applies the end position of the `narrow` column in the Porsche Grid.     |
| `--pds-grid-narrow-span-one-half`   | `.col-(pds-grid-narrow-span-one-half)`          | Applies a half span within the `narrow` area of the Porsche Grid.        |
