# Grid

<TableOfContents headings={[]} />

| CSS Variable                        | Description                                                   |
| ----------------------------------- | ------------------------------------------------------------- |
| `--pds-grid-narrow-column-start`    | The start position of the `narrow` Porsche Grid column.       |
| `--pds-grid-narrow-column-end`      | The end position of the `narrow` Porsche Grid column.         |
| `--pds-grid-narrow-span-one-half`   | The span of one half within the `narrow` Porsche Grid area.   |
| `--pds-grid-basic-column-start`     | The start position of the `basic` Porsche Grid column.        |
| `--pds-grid-basic-column-end`       | The start position of the `basic` Porsche Grid column.        |
| `--pds-grid-basic-span-one-half`    | The span of one half within the `basic` Porsche Grid area.    |
| `--pds-grid-basic-span-one-third`   | The span of one third within the `basic` Porsche Grid area.   |
| `--pds-grid-basic-span-two-thirds`  | The span of two half within the `basic` Porsche Grid area.    |
| `--pds-grid-extended-column-start`  | The start position of the `extended` Porsche Grid column.     |
| `--pds-grid-extended-column-end`    | The end position of the `extended` Porsche Grid column.       |
| `--pds-grid-extended-span-one-half` | The span of one half within the `extended` Porsche Grid area. |
| `--pds-grid-wide-column-start`      | The start position of the `wide` Porsche Grid column.         |
| `--pds-grid-wide-column-end`        | The end position of the `wide` Porsche Grid column.           |
| `--pds-grid-full-column-start`      | The start position of the `full` Porsche Grid column.         |
| `--pds-grid-full-column-end`        | The end position of the `full` Porsche Grid column.           |
