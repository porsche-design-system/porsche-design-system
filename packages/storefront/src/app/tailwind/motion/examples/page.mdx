## Duration

The Porsche Design System Tailwind theme adds custom CSS Variables for durations `--pds-duration-*`, which can be used
in tailwind classes with the custom property syntax `.duration-(--pds-duration-*)`.

## Easing

The Porsche Design System Tailwind theme customizes the Tailwind variable `--ease-*`, which affect multiple Tailwind
classes including `.ease-*` and others.
