## Tokens

| Token           | Description                                                                 | Value           |
| --------------- | --------------------------------------------------------------------------- | --------------- |
| `breakpointXs`  | Holds the `xs` breakpoint value, usable with `px` (**{breakpointXs}px**).   | {breakpointXs}  |
| `breakpointSm`  | Holds the `sm` breakpoint value, usable with `px` (**{breakpointSm}px**).   | {breakpointSm}  |
| `breakpointMd`  | Holds the `md` breakpoint value, usable with `px` (**{breakpointMd}px**).   | {breakpointMd}  |
| `breakpointLg`  | Holds the `lg` breakpoint value, usable with `px` (**{breakpointLg}px**).   | {breakpointLg}  |
| `breakpointXl`  | Holds the `xl` breakpoint value, usable with `px` (**{breakpointXl}px**).   | {breakpointXl}  |
| `breakpoint2Xl` | Holds the `2xl` breakpoint value, usable with `px` (**{breakpoint2Xl}px**). | {breakpoint2Xl} |
