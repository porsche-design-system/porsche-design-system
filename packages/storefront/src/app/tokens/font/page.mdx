## Tokens

### Font

| Token            | Description                                                                               | Value            |
| ---------------- | ----------------------------------------------------------------------------------------- | ---------------- |
| `fontFamily`     | Holds the **Porsche Next** font family along with fallback fonts.                         | {fontFamily}     |
| `fontLineHeight` | Holds a dynamic default line height specifically optimized for the Porsche Next typeface. | {fontLineHeight} |

### Font Size

| Token         | Description                                                                               | Value         |
| ------------- | ----------------------------------------------------------------------------------------- | ------------- |
| `fontSize2Xs` | Holds the **2x-small** font size and line height optimized for the Porsche Next typeface. | {fontSize2Xs} |
| `fontSizeXs`  | Holds the **x-small** font size and line height optimized for the Porsche Next typeface.  | {fontSizeXs}  |
| `fontSizeSm`  | Holds the **small** font size and line height optimized for the Porsche Next typeface.    | {fontSizeSm}  |
| `fontSizeMd`  | Holds the **medium** font size and line height optimized for the Porsche Next typeface.   | {fontSizeMd}  |
| `fontSizeLg`  | Holds the **large** font size and line height optimized for the Porsche Next typeface.    | {fontSizeLg}  |
| `fontSizeXl`  | Holds the **x-large** font size and line height optimized for the Porsche Next typeface.  | {fontSizeXl}  |
| `fontSize2Xl` | Holds the **2x-large** font size and line height optimized for the Porsche Next typeface. | {fontSize2Xl} |

### Font Weight

| Token                | Description                                                                  | Value                |
| -------------------- | ---------------------------------------------------------------------------- | -------------------- |
| `fontWeightRegular`  | Holds the **regular** font weight optimized for the Porsche Next typeface.   | {fontWeightRegular}  |
| `fontWeightSemiBold` | Holds the **semi-bold** font weight optimized for the Porsche Next typeface. | {fontWeightSemiBold} |
| `fontWeightBold`     | Holds the **bold** font weight optimized for the Porsche Next typeface.      | {fontWeightBold}     |
