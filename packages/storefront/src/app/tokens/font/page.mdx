## Tokens

### Font Family

| Token             | Description                                                       | Value             |
| ----------------- | ----------------------------------------------------------------- | ----------------- |
| `fontPorscheNext` | Holds the **Porsche Next** font family along with fallback fonts. | {fontPorscheNext} |

### Font Size

| Token          | Description                                                                               | Value          |
| -------------- | ----------------------------------------------------------------------------------------- | -------------- |
| `typescale2Xs` | Holds the **2x-small** font size and line height optimized for the Porsche Next typeface. | {typescale2Xs} |
| `typescaleXs`  | Holds the **x-small** font size and line height optimized for the Porsche Next typeface.  | {typescaleXs}  |
| `typescaleSm`  | Holds the **small** font size and line height optimized for the Porsche Next typeface.    | {typescaleSm}  |
| `typescaleMd`  | Holds the **medium** font size and line height optimized for the Porsche Next typeface.   | {typescaleMd}  |
| `typescaleLg`  | Holds the **large** font size and line height optimized for the Porsche Next typeface.    | {typescaleLg}  |
| `typescaleXl`  | Holds the **x-large** font size and line height optimized for the Porsche Next typeface.  | {typescaleXl}  |
| `typescale2Xl` | Holds the **2x-large** font size and line height optimized for the Porsche Next typeface. | {typescale2Xl} |

### Line Height

| Token           | Description                                                                               | Value           |
| --------------- | ----------------------------------------------------------------------------------------- | --------------- |
| `leadingNormal` | Holds a dynamic default line height specifically optimized for the Porsche Next typeface. | {leadingNormal} |

### Font Weight

| Token                | Description                                                                  | Value                |
| -------------------- | ---------------------------------------------------------------------------- | -------------------- |
| `fontWeightNormal`   | Holds the **normal** font weight optimized for the Porsche Next typeface.    | {fontWeightNormal}   |
| `fontWeightSemibold` | Holds the **semi-bold** font weight optimized for the Porsche Next typeface. | {fontWeightSemibold} |
| `fontWeightBold`     | Holds the **bold** font weight optimized for the Porsche Next typeface.      | {fontWeightBold}     |
