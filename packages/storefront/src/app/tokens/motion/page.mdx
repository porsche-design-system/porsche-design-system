## Tokens

### Duration

| Token        | Description                                             | Value        |
| ------------ | ------------------------------------------------------- | ------------ |
| `durationSm` | Holds a **short** motion/transition duration value.     | {durationSm} |
| `durationMd` | Holds a **moderate** motion/transition duration value.  | {durationMd} |
| `durationLg` | Holds a **long** motion/transition duration value.      | {durationLg} |
| `durationXl` | Holds a **very long** motion/transition duration value. | {durationXl} |

### Ease

| Token       | Description                                       | Value       |
| ----------- | ------------------------------------------------- | ----------- |
| `easeInOut` | Holds an **in-out** `transition-timing-function`. | {easeInOut} |
| `easeIn`    | Holds an **in** `transition-timing-function`.     | {easeIn}    |
| `easeOut`   | Holds an **out** `transition-timing-function`.    | {easeOut}   |
