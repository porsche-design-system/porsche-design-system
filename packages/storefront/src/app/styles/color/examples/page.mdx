# Color

<TableOfContents headings={['Text Color', 'Background Color', 'Border Color']} />

The Porsche Design System uses **CSS variables** to define colors. These variables are required as global styles because
all components depend on them.

They are included in the main CSS file:

```css
@import '@porsche-design-system/components-{js|angular|react|vue}/index.css';
```

Alternatively, you can import only the color variables:

```css
@import '@porsche-design-system/components-{js|angular|react|vue}/variables.css';
```

By default, color tokens use the light theme. To enable the dark theme or automatic theme switching based on system
settings, add the `.light`, `.dark` or `.auto` CSS class to the `<html>` element or any HTML element. This applies the
selected theme to all child elements.
