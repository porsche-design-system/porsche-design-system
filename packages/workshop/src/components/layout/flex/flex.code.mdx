# Flex

## Introduction

The Porsche UI Kit flex layout system is based on standard css flexbox browser behaviour and can be controlled by changing class names/properties on the flex container and flex item. It can be used to quickly layout standard, css-prefixed content blocks or components. It does not replace [Grid](#/layout/grid) component which should be used to define basic page structures.

### Standard (default)
Initialize standard flexbox container to define flex context.
If using default property, no extra class/prop declaration is needed due to standard browser behaviour.

<Example noTheme>
  {(theme) => {
    const itemStyle = (color) => {
      return {
        backgroundColor: color,
        color: "white",
        width: "200px"
      }
    }
    return (
      <React.Fragment>
        <Flex>
          <Flex.Item>
            <Text align="center" style={itemStyle("deepskyblue")}>
              1
            </Text>
          </Flex.Item>
          <Flex.Item>
            <Text align="center" style={itemStyle("lightskyblue")}>
              2
            </Text>
          </Flex.Item>
        </Flex>
        <Flex>
          <Flex.Item>
            <Text align="center" style={itemStyle("deepskyblue")}>
              1
            </Text>
          </Flex.Item>
          <Flex.Item>
            <Text align="center" style={itemStyle("lightskyblue")}>
              2
            </Text>
          </Flex.Item>
        </Flex>
      </React.Fragment>
    )
  }}
</Example>

### Flex wrap

The flex wrap property is used to force flex items to stay in line independently of the flex container width, or to flow in multiple lines forced by the flex container width.

#### Nowrap (default)

If using default property, no extra class/prop declaration is needed due to standard browser behaviour.

<Example noTheme>
  {(theme) => {
    const itemStyle = (color) => {
      return {
        backgroundColor: color,
        color: "white",
        width: "200px",
        textAlign: "center"
      }
    }
    return (
      <React.Fragment>
        <Flex>
          {Array.apply(0, Array(9)).map(function(x, i) {
            return (
              <Flex.Item key={i}>
                <Text align="center" style={itemStyle("deepskyblue")}>
                  {i + 1}
                </Text>
              </Flex.Item>
            )
          })}
        </Flex>
      </React.Fragment>
    )
  }}
</Example>

### Flex child spacing

Use `gap` to apply equal horizontal spacing to all children of the container. You have to take care of vertical spacing yourself. This should not be an alternative to the global grid system/component. All values of standard spacing component can be used.

**Static**

`"auto" | 0 | 4 | 8 | 12 | 16 | 20 | 24 | 32 | 40 | 48 | 56 | 64 | 72 | 80`

**Responsive**

`"a" | "b" | "c" | "d" | "e" | "f" | "g"`

* `{p}-flex__child--align-self-end` => item is at the end line of the cross axis
* `{p}-flex__child--align-self-center` => item is centered on the cross axis
* `{p}-flex__child--align-self-baseline` => item is centered on the (text-) baseline of cross axis
* `{p}-flex__child--align-self-end-{bp}` => example of breakpoint specific behaviour

### Flex item grow

Flexbox default behaviour on how the childrens widths are rendered (stretched) can be overwritten with the following class names.

It handles how to grow an item based on the space that is left to fulfill the parents width.

#### Grow - 0 (default)

If using default property, no extra class/prop declaration is needed due to standard browser behaviour.

  {(theme) => {

  }}
</Example>
