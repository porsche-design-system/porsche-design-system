# Flex

## Introduction

The Porsche UI Kit flex layout system is based on native css flex behaviours and can be controlled by class names.

### Flex standard

<Example noTheme>
  {(theme) => {
    const itemStyle = (color) => {
      return {
        backgroundColor: color,
        padding: "12px",
        color: "white",
        width: "200px"
      }
    }
    return (
      <React.Fragment>
        <Flex>
          <Flex.Item>
            <Text align="center" style={itemStyle("DeepSkyBlue")}>
              1
            </Text>
          </Flex.Item>
          <Flex.Item>
            <Text align="center" style={itemStyle("LightSkyBlue")}>
              2
            </Text>
          </Flex.Item>
        </Flex>
        <Flex>
          <Flex.Item>
            <Text align="center" style={itemStyle("DeepSkyBlue")}>
              1
            </Text>
          </Flex.Item>
          <Flex.Item>
            <Text align="center" style={itemStyle("LightSkyBlue")}>
              2
            </Text>
          </Flex.Item>
        </Flex>
      </React.Fragment>
    )
  }}
</Example>

### Flex responsive behaviour (base: flex, l: flex-inline):

<Example noTheme>
