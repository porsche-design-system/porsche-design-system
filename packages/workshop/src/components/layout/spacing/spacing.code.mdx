# Spacing

## Introduction

A component to add margins and paddings to components. If this component has only one child, those classes are added directly to the child using the className prop to avoid unnecessary wrapper divs. This only works if the child component supports className (which it totally should anyway) and if the number of children isn't dynamic and doesn't change. If this is not the case, set the "wrap" property to manually create a wrapper div.
