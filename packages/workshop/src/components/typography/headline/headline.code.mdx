# Headline

## Introduction
All text stylings can be set by prop/classname (see examples below). If you can't add a text class to an element, there are also SASS mixins for predefined text and SASS variables for generic text.

### Predefined SASS mixins
```
// headlines
@include p-text-large-title
@include p-text-headline-1
@include p-text-headline-2
@include p-text-headline-3
@include p-text-headline-4
@include p-text-headline-5
```

## Headlines

### Large-title

<Example noTheme={false} className="sg-example-text">
  {(theme) => {
    const setTheme = theme === "inverted" ? true : false
    return (
      <React.Fragment>
        <Headline type="large-title" level="1" inverted={setTheme}>Lorem ipsum dolor sit amet</Headline>
      </React.Fragment>
    )
  }}
</Example>
