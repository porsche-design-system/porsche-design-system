In vanilla js even if hidden is explicitly defined as @Prop in the component hidden="false" won't work! For all other frameworks hidden="false" should work since the typing allows a boolean value here. Option should be visible (no hidden attribute) Option shouldn't be visible (hidden attribute without value) Option shouldn't be visible (hidden attribute with boolean true) Option shouldn't be visible (hidden attribute with boolean false) Button should be visible (no hidden attribute) Button shouldn't be visible (hidden attribute without value) Button shouldn't be visible (hidden attribute with boolean true) Button shouldn't visible (hidden attribute with boolean false)
