# Loader

## Introduction
Basic animated loader to visualize loading states, e.g. page loading, form validation, etc. The Loader component sizes are predefined.

### X-small

<Example noTheme={false}>
  {(theme) => {
    const setTheme = theme === "inverted" ? true : false
    return (
      <React.Fragment>
        <p-loader size="x-small" inverted={setTheme}></p-loader>
      </React.Fragment>
    )
}}
</Example>
