# Typography

## Introduction
Text component can be categorized into 2 different use cases:

1. Predefined headlines with automated responsive sizing to fit into all major breakpoints
2. Generic text sizes which are fixed and do not respond to different viewports

## Headline

### Predefined SASS mixins
```
@include p-text-large-title
@include p-text-headline-1
@include p-text-headline-2
@include p-text-headline-3
@include p-text-headline-4
@include p-text-headline-5
```

### Large-title

<Example noTheme={false} className="sg-example-text">
  {(theme) => {
    return (
      <React.Fragment>
        <p-headline type="large-title" level="1" theme={theme}>Lorem ipsum dolor sit amet</p-headline>
      </React.Fragment>
    )
  }}
</Example>

## Text

### Predefined copy text SASS mixins
```
@include p-text-copy
@include p-text-small
```

### Generic size SASS variables
```
$p-text-size-12: rem(12px);
$p-text-size-16: rem(16px);
$p-text-size-20: rem(20px);
$p-text-size-24: rem(24px);
$p-text-size-28: rem(28px);
$p-text-size-30: rem(30px);
$p-text-size-32: rem(32px);
$p-text-size-36: rem(36px);
$p-text-size-42: rem(42px);
$p-text-size-44: rem(44px);
$p-text-size-48: rem(48px);
$p-text-size-52: rem(52px);
$p-text-size-60: rem(60px);
$p-text-size-62: rem(62px);
$p-text-size-72: rem(72px);
$p-text-size-84: rem(84px);
```

### Font weights SASS variables
```
$p-font-weight-thin: 200;
$p-font-weight-regular: 400;
$p-font-weight-bold: 600;
```
