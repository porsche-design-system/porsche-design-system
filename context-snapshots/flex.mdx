# Flex (Component: p-flex)

The Porsche Design System flex layout system is based on standard CSS Flexbox browser behaviour and can be controlled by
the properties of the `p-flex` container and `p-flex-item`. It can be used to quickly layout standard content blocks or
components.

<Notification heading="Deprecation hint" heading-tag="h2" state="error">
  This component is deprecated and will be removed with the next major release.
  In general, please use native <Link href="https://css-tricks.com/snippets/css/a-guide-to-flexbox">CSS Flex</Link> instead for better performance and more standardized layout technique.
</Notification>
